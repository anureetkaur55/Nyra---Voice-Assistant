# Nyra---Voice-Assistant
Nyra is an AI-powered voice assistant web app with speech input, spoken replies, OpenAI/Hugging Face answers, Wikipedia search, YouTube, Google and a Node.js backend.

## Features

- Voice input
- Voice output
- OpenAI answers
- Hugging Face answers
- Wikipedia search
- YouTube search
- Google open command
- Time command

## Setup

Install Node.js 20 or newer.

Create `.env` from `.env.example`:

```env
OPENAI_API_KEY=your_openai_api_key_here
HF_TOKEN=your_huggingface_token_here
HF_MODEL=deepseek-ai/DeepSeek-V3.2:novita
PORT=3000
